# 健康の新常識ノート

かつて常識とされ、いまは見直された健康の知識を「旧常識 → 現在の見解 → 出典」の形で、1項目2分以内で読めるようにまとめたサイトです。あわせて、肩こり・腰痛・膝の定番体操を「学会・公的機関が出所のものを1症状につき1つだけ」に絞って紹介しています。

**公開ページ:** https://yuki198.github.io/kenko-shinjoshiki-note/

## 方針

- 公的機関・学会・査読論文など、検証可能な出典のない項目は載せない
- 各項目に「最終確認日」と「エビデンスの確かさ」（公的基準に反映済み／専門機関の見解／研究が続いている）を表示する
- 見解が割れているテーマは、割れていると正直に書く
- 医師による診断・治療が必要な病気・症状は扱わない

本サイトは一般的な健康情報の紹介であり、医療上の助言ではありません。
